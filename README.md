# Channel Estimation in Uplink Multi-User Scenario using OTFS Modulation

Code for our paper at the 2025 ITG Conference on Systems, Communications  and Coding (SCC).

[Paper (arXiv)](https://arxiv.org/abs/2404.11328)

## Abstract

This work investigates channel estimation techniques for the orthogonal time frequency space (OTFS) modulation scheme. It studies the orthogonal matching pursuit algorithm with and without side information, proposes an efficient pilot and data placement strategy for multi-user impulse pilot-based estimation, and evaluates the performance of the proposed estimation methods across various multi-user scenarios.

## What is in this repository

- MATLAB implementation of OMP-based channel estimation
- The proposed pilot/data placement scheme for multi-user scenarios
